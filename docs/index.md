# Recoll for Windows

Free Windows installer and prebuilt binaries for Recoll.

## Download

- [Latest Windows installer](https://github.com/alarmz/recoll/releases/latest)
- Repository: [alarmz/recoll](https://github.com/alarmz/recoll)

## What is included

- Windows x64 installer
- Prebuilt binaries
- Source code
- Winget manifest files

## Search terms this page is meant to help with

- Recoll Windows
- Recoll Windows download
- Recoll Windows installer
- Recoll Windows binary
- Recoll setup.exe
- Free Recoll Windows build

## Install

1. Download the latest `recoll-setup-*.exe` from Releases.
2. Run the installer.
3. Start Recoll from the Start menu.

## Why this exists

Recoll is well known on Linux, but Windows users often have difficulty finding a free and direct binary download. This repository exists to make that easier.
