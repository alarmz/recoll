# Recoll for Windows 1.43.13

Recoll for Windows 1.43.13 is now available.

This release provides a free, community-maintained Windows installer for Recoll, the open source desktop full-text search tool. If you were looking for a direct Recoll Windows download without building from source, this is the release to use.

## Download

- Windows installer (64-bit): `recoll-Next-win64-setup.exe`

## Why this release matters

Recoll is well known on Linux, but Windows users often have trouble finding a straightforward binary installer. This release makes Recoll easier to install on Windows with a ready-to-run setup package.

## What is Recoll?

Recoll is an open source desktop full-text search application. It can search inside document contents as well as file names, making it useful for finding PDFs, Office documents, text files, archives, and more.

## Included in this release

- Windows x64 installer
- Prebuilt Windows binaries
- Public source repository
- Package manifests for Scoop and winget workflow integration

## Installation

### Option 1: Direct install

1. Download `recoll-Next-win64-setup.exe`
2. Run the installer
3. Start Recoll from the Start menu

### Option 2: Scoop

```powershell
scoop bucket add alarmz https://github.com/alarmz/recoll
scoop install alarmz/recoll
```

### Option 3: winget

This repository includes a `winget` manifest prepared for submission to the Windows Package Manager community repository.
