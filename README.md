## English

### Recoll — Desktop Full-Text Search for Linux & Windows

Recoll is a desktop full-text search tool. It finds keywords inside documents as well as file names.

### Download for Windows (Free)

**[Download Recoll Windows Installer (64-bit)](https://github.com/alarmz/recoll/releases/latest)** — Free, no compilation needed.

- Built with Qt 6.8.2
- 64-bit Windows (x64) installer
- Same full-text search power as the Linux version

> Looking for the Linux version? Most distributions have Recoll in their package manager.

### Features

* Available for Linux and MS Windows.
* A WEB front-end with preview and download features can replace or supplement the GUI for remote use.
* Searches most document formats. You may need external applications for text extraction.
* Can reach any storage place: files, archive members, email attachments, transparently handling decompression.
* One click will open the document inside a native editor or display an even quicker text preview.
* The software is free, open source, and licensed under the GPL.

For more detail, see the [features page](https://www.recoll.org/pages/features.html) or the [online documentation](https://www.recoll.org/pages/documentation.html).

### Installation

#### Windows
Download the installer from the [Releases page](https://github.com/alarmz/recoll/releases/latest) and run it. No additional setup required.

#### Linux
Most distributions feature prebuilt packages for Recoll. Install via your package manager (e.g. `apt install recoll`, `dnf install recoll`, `pacman -S recoll`).

If you need to build from source, [the instructions are here](https://www.recoll.org/usermanual/usermanual.html#RCL.INSTALL.BUILDING).
